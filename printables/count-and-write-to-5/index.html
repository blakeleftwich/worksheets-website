How to Use These Worksheets

Students count each group of illustrated objects and write the correct number in the space provided. With quantities limited to 1 through 5, these worksheets are perfect for young learners who are just beginning to connect counting with number writing.

In the Classroom

Use these as introductory counting worksheets for Pre-K and early Kindergarten students. They work well for morning warm-ups, math centers, or independent practice. The small number range helps build confidence and accuracy before moving to larger numbers.

At Home

Count small groups of everyday objects with your child — fingers on a hand, toys on a table, or snacks on a plate. Then have them practice writing the number. Keeping quantities small helps young learners focus on forming each numeral correctly.

Tips for Success

  • Encourage touching each object while counting out loud
  • Model correct number formation before students write independently
  • Use the fun images to keep students engaged and motivated
  • Use the answer key for immediate feedback

Common Core Standards Alignment

These worksheets support the following Common Core State Standards for Mathematics:

  • K.CC.B.4a — When counting objects, say the number names in the standard order, pairing each object with one and only one number name
  • K.CC.B.5 — Count to answer “how many?” questions about as many as 20 things arranged in various configurations
  • K.CC.A.3 — Write numbers from 0 to 20; represent a number of objects with a written numeral
